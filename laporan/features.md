# Features this App

Nama Aplikasi :Absensi Pegawai

Kelompok :Disduk

## Pengguna

1. Admin
2. Pegawai


## Menu

**Untuk Pengguna 1(Admin)**

- login
- user
- users
- mengelola data pegawai
- tambah data
- pengajuan izin
- jumlah pegawai
- perubahan data
- logout
- backup db

**Untuk Pengguna 2(Pegawai)**

- login
- user
- memasukan data 
- profile (ubah username & password)
- mengubah data pribadi
- pengajuan izin
- Logout



## Fitur

**Untuk Pengguna 1(Admin)**

1. Login

2. user

3. Profile

4. perubahan data 
- user-tambah.php (v)
- user-hapus.php (p)

6. Logout

- logout.php (P)

**Untuk Pengguna 2(Pegawai)**

1. Login

- login.php 1.1.1 (V)
- cek-login.php 1.1.2 (P)

2. user

- user-ubah.php (V)
- user-perbaharui.php (P)

### Keterangan

(V) : View, Untuk file yang menghasilkan tampilan input / output.

(P) : Process, Untuk file tidak menghasilkan tampilan (Seperti database.php, proses-hapus.php).
